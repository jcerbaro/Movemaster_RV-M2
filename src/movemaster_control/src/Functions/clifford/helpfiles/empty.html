Clifford Function Reference

empty

This function makes it possible to use the dotted notation clifford.empty to create an empty array.
(Clifford overloading of standard MATLAB® function)

Description

clifford.empty creates an empty multivector array. The size of the array may be specified using the same parameter profiles as in MATLAB® and additionally the class of the empty components may be specified with a string parameter which must come last.

Examples

>> x = clifford.empty
 
x = [] Cl(0,2) multivector
     
>> x = clifford.empty([1 0], 'int8')
 
x = Empty Cl(0,2) multivector matrix: 1-by-0
