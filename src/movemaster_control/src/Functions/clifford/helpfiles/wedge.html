Clifford Function Reference

wedge

Clifford wedge product

Description

wedge(x,y) computes the wedge product between two elements x and y. The function takes an arbitrary number of parameters, because the wedge product is associative. Thus wedge(x, y, z) == wedge(wedge(x, y), z) as demonstrated in the second example below.

Examples

>> clifford_signature(1,1)
>> x = clifford(1,2,3,4)
 
x =
 
   1.0000 e0
 + 2.0000 e1 + 3.0000 e2
 + 4.0000 e12
 
>> y = clifford(5,6,7,8)
 
y =
 
   5.0000 e0
 + 6.0000 e1 + 7.0000 e2
 + 8.0000 e12
 
>> wedge(x,y)
 
ans =
 
   5.0000 e0
 + 16.0000 e1 + 22.0000 e2
 + 24.0000 e12

Second example:
 
>> clifford_signature(2,1)
>> x = randm; y = randm; z = randm;
>> wedge(x, y, z) == wedge(wedge(x, y), z)

ans =

  logical

   1
