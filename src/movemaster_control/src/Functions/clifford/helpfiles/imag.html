Clifford Function Reference

imag

Imaginary part of a complex multivector
(Clifford overloading of standard MATLAB® function)

Syntax

Y = imag(X)

Description

imag(X) returns the imaginary part of a complex multivector If X is a real multivector imag(X) returns zero.
