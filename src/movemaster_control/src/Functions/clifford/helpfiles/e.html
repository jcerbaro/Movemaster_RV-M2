Clifford Function Reference

e0, e1 etc

Clifford basis blades
(Clifford overloading of standard MATLAB® function)

Description

These functions represent the clifford basis blades e0, e1, e12, ..., e123. They are implemented as parameterless functions. A limited set of these are distributed with the toolbox. More can be created by the user if needed - see the file clifford_parameterless_functions which is a script that can be adapted by editing the file parameterless.lst.

Only the lexically ordered permutations of these blades are provided as functions, but others can be created by calling the Clifford constructor with a string parameter, e.g. clifford('e312'). Equally, this method can be used to create basis blades for which a parameterless function file does not exist.

Examples

>> clifford_signature(3,3)
>> e1234
 
ans = 1.0000 e1234 

>> clifford('e21') + clifford('e431')

ans = - 1.0000 e12   
      - 1.0000 e134
