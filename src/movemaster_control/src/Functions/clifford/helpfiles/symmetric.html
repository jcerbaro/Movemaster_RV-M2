Clifford Function Reference

symmetric

Extracts those grades of a multivector which do not change sign under reversion.

Description

p = symmetric(m) returns in p those grades of multivector m which do not change sign under reversion.

Examples

>> clifford_signature(3,1)
>> m = randm
 
m =   0.0809 e0  
    + 0.2758 e1   - 0.3397 e2   + 0.1297 e3   + 0.0479 e4  
    - 0.1967 e12  - 0.0652 e13  + 0.0515 e14  + 0.5382 e23  + 0.4165 e24  - 0.2030 e34 
    + 0.4565 e123 + 0.1091 e124 - 0.0095 e134 + 0.1075 e234
    - 0.0308 e1234
 
>> symmetric(m)
 
ans =    0.0809 e0  
       + 0.2758 e1   - 0.3397 e2   + 0.1297 e3   + 0.0479 e4  
       - 0.0308 e1234
 
>> reverse(m)
 
ans =   0.0809 e0  
      + 0.2758 e1   - 0.3397 e2   + 0.1297 e3   + 0.0479 e4  
      + 0.1967 e12  + 0.0652 e13  - 0.0515 e14  - 0.5382 e23  - 0.4165 e24  + 0.2030 e34 
      - 0.4565 e123 - 0.1091 e124 + 0.0095 e134 - 0.1075 e234
      - 0.0308 e1234
