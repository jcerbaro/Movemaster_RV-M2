Clifford Function Reference

coefficients

Returns the coefficients of the multivector in a cell array.

Description

c = coefficients(m) returns in c the numeric coefficients of the multivector m as a cell array. Some of the cells may be empty, corresponding to coefficients with zero values.

Examples

>> m = randn(2) .* e2
 
m = 2x2 Cl(0,2) multivector matrix
 
>> c = coefficients(m)

c = 1×4 cell array

    {0×0 double}    {0×0 double}    {2×2 double}    {0×0 double}

>> c{3}

ans = 1.0347   -0.3034
      0.7269    0.2939
