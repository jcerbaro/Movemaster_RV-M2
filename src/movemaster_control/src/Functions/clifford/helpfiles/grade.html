Clifford Function Reference

grade

Extracts the y-th grade component of a clifford multivector. The result may be empty (example: grade(e1, 2) = []).

Description

[g, e] = grade(x,y) returns in g the elements of multivector x, which are part of the grade, y. The second return result, if required, shows whether the result in g is empty, as a true or false value.

Examples

>> x = randm
 
x =  0.3445 e0  
   + 0.0708 e1   + 0.0480 e2   + 0.3854 e3   - 0.1953 e4  
   + 0.1691 e12  + 0.2027 e13  - 0.0592 e14  + 0.0524 e23  - 0.2830 e24  - 0.2787 e34 
   + 0.0255 e123 + 0.1753 e124 + 0.6276 e134 - 0.1619 e234
   + 0.0455 e1234
 
>> b = bivector(x)
 
b = 0.1691 e12  + 0.2027 e13  - 0.0592 e14  + 0.0524 e23  - 0.2830 e24  - 0.2787 e34 
 
>> [v, e] = grade(x, 1)
 
v = 0.0708 e1   + 0.0480 e2   + 0.3854 e3   - 0.1953 e4  
 
e = logical 0

>> [v, e] = grade(b, 1)
 
v = [] Cl(0,4) multivector

e = logical 1
