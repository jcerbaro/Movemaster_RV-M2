Clifford Function Reference

sqrt

Square root
(Clifford overloading of standard MATLAB® function)

Syntax

Y = sqrt(X)

Description

sqrt(X) computes the square root of each element of the multivector array X.

Pending a better algorithm the current implementation uses the MATLAB® sqrtm function on the isomorphic matrix of each element of X.
