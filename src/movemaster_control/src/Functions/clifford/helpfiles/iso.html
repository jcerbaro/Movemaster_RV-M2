Clifford Function Reference

iso

Convert multivector array to isomorphic array.

Description

iso(m) maps the multivector array m to an array of reals (or complex values if the coefficients of the multivector were complex). On return the current Clifford algebra remains that of m.

iso(m, p, q) maps the multivector array m to a multivector array in algebra with signature (p, q), if this mapping is possible. On return the current Clifford algebra has signature (p, q), that is, a call on the iso function has a side-effect of altering the current algebra.
