Clifford Function Reference

randm

Make random clifford multivector.

Description

randm creates random multivectors of unit norm in the current algebra.

It accepts parameters similar to the MATLAB® functions rand and randn to specify the dimensions of the multivector array (default 1-by-1).

In addition, it accepts an optional final pair of parameters in the form 'sparse', k where k is a numeric value between 1/m and 1, where m is the number of elements in a multivector of the current algebra. In this case, the result is a multivector with some empty elements (in fact (1 - k) * m empty elements).

Examples

>> randm(2)
 
ans =
 
     2x2 Cl(0,2) multivector matrix
 
>> show(ans)
 
ans =
 
e0 *
 
   -0.6475    0.0383
    0.7220   -0.3985

+ e1 *
 
    0.5411    0.8745
   -0.4002   -0.1258

+ e2 *
 
   -0.3898   -0.3179
    0.5512    0.0039

+ e12 *
 
    0.3687   -0.3643
    0.1212   -0.9085

>> randm('sparse', 0.5)
 
ans =
 
   0.9444 e0
 + 0.3288 e12
