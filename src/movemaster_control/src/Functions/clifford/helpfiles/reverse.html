Clifford Function Reference

reverse

Computes the reverse of a clifford multivector.

Description

reverse(x) calculates the reverse of multivector x. "To reverse x|| → −x|| x′ = −x|| + x⊥ = −a−1 a x|| + a−1 a x⊥ = −a−1 x||a − a−1 x⊥a = −a−1(x|| + x⊥)a = −a−1 xa"[1]

The reverse operates on any basis element to give the basis element (and sign) that results from reversing the order of the lexical indices. For example the reverse of e134 is e431. This may be verified using the clifford constructor as shown in the second example below.

The grade involution, reverse and clifford conjugate form a closed set under composition: any two composed give the same result as the third.

Examples

>> clifford_signature(1,1)
>> x = clifford(1,2,3,4)
 
x =
 
   1.0000 e0
 + 2.0000 e1 + 3.0000 e2
 + 4.0000 e12
 
>> reverse(x)
 
ans =
 
   1.0000 e0
 + 2.0000 e1 + 3.0000 e2
 - 4.0000 e12
Second example, showing the fundamental concept of the reverse: 
>> clifford_signature(3,1)
>> reverse(e134)
 
ans = - 1.0000 e134
 
>> clifford('e431')
 
ans = - 1.0000 e134
