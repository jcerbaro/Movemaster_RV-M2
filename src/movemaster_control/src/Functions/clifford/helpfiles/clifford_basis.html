Clifford Function Reference

clifford_basis

Vector of Clifford basis elements.

Description

clifford_basis returns a column vector containing all the basis elements of the current algebra, starting with the scalar and ending with the pseudoscalar.

Examples

>> clifford_signature(0,2)
>> show(clifford_basis)
 
e0 *
 
     1
     0
     0
     0

+ e1 *
 
     0
     1
     0
     0

+ e2 *
 
     0
     0
     1
     0

+ e12 *
 
     0
     0
     0
     1
