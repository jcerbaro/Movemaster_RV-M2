Clifford Function Reference

sqrtm

Matrix square root
(Clifford overloading of standard MATLAB® function)

Syntax

Y = sqrtm(X)

Description

sqrtm(X) computes the matrix square root of a multivector matrix. The algorithm used is based on calling the MATLAB® sqrtm function on an isomorphic matrix, and is therefore not accurate for larger matrices. The accuracy may be checked by squaring the result using ^.
