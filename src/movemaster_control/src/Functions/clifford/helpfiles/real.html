Clifford Function Reference

real

Real part of a complex multivector
(Clifford overloading of standard MATLAB® function)

Syntax

Y = real(X)

Description

real(X) returns the real part of a complex multivector.

The concept of the real part of a multivector should not be confused with the scalar part (returned by the function scalar).
