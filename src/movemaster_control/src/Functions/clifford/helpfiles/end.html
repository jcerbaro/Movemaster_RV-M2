Clifford Function Reference

end

End indexing for multivector arrays
(Clifford overloading of standard MATLAB® function)

Syntax

e = end(a, k, n)

Description

end overloads a fundamental MATLAB® operation for Clifford multivector arrays in order to make possible indexing such as A(2:end, 2:end). It is not normally called directly as a function.
