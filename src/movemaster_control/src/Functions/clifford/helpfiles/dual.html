Clifford Function Reference

dual

Computes the dual of a multivector.

Description

dual(x) computes the dual of a multivector, x. The dual is defined as the product of the inverse of the pseudoscalar (on the left) with the multivector x, as shown in the example below.

Examples

>> clifford_signature(3,0)
>> m = randm
 
m =
 
 - 0.1792 e0 
 + 0.1462 e1  + 0.2207 e2  + 0.5109 e3 
 - 0.0579 e12 - 0.6382 e13 - 0.2506 e23
 + 0.4043 e123
 
>> dual(m), inv(e123) .* m
 
ans =
 
   0.4043 e0 
 - 0.2506 e1  + 0.6382 e2  - 0.0579 e3 
 - 0.5109 e12 + 0.2207 e13 - 0.1462 e23
 + 0.1792 e123
 
 
ans =
 
   0.4043 e0 
 - 0.2506 e1  + 0.6382 e2  - 0.0579 e3 
 - 0.5109 e12 + 0.2207 e13 - 0.1462 e23
 + 0.1792 e123
 
>> inv(e123)
 
ans = - 1.0000 e123
