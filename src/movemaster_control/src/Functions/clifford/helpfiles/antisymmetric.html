Clifford Function Reference

antisymmetric

Extracts those grades of a multivector which change sign under reversion.

Description

p = antisymmetric(m) returns in p those grades of multivector m which change sign under reversion.

Examples

>> clifford_signature(3,1)
>> m = randm
 
m =  - 0.0310 e0  
     + 0.3718 e1   + 0.3517 e2   + 0.3537 e3   + 0.1676 e4  
     - 0.3014 e12  + 0.1790 e13  + 0.4069 e14  + 0.1220 e23  + 0.2583 e24  + 0.1814 e34 
     - 0.0757 e123 + 0.0734 e124 - 0.1965 e134 + 0.2217 e234
     - 0.2863 e1234
 
>> antisymmetric(m)
 
ans =  - 0.3014 e12  + 0.1790 e13  + 0.4069 e14  + 0.1220 e23  + 0.2583 e24  + 0.1814 e34 
       - 0.0757 e123 + 0.0734 e124 - 0.1965 e134 + 0.2217 e234
 
>> reverse(m)
 
ans =  - 0.0310 e0  
       + 0.3718 e1   + 0.3517 e2   + 0.3537 e3   + 0.1676 e4  
       + 0.3014 e12  - 0.1790 e13  - 0.4069 e14  - 0.1220 e23  - 0.2583 e24  - 0.1814 e34 
       + 0.0757 e123 - 0.0734 e124 + 0.1965 e134 - 0.2217 e234
       - 0.2863 e1234
