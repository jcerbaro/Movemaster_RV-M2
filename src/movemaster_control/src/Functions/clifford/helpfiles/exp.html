Clifford Function Reference

exp

Exponential
(Clifford overloading of standard MATLAB® function)

Syntax

Y = exp(X)

Description

exp(X) computes the exponential of each element of its argument.

Examples

>> clifford_signature(0,3)
>> exp(1 + e1 + e12)
 
ans =
 
   0.4239 e0 
 + 1.8986 e1 
 + 1.8986 e12
