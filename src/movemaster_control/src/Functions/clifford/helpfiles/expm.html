Clifford Function Reference

expm

Matrix exponential
(Clifford overloading of standard MATLAB® function)

Syntax

Y = expm(X)

Description

expm(X) computes the matrix exponential for a multivector matrix. The algorithm used is based on calling the MATLAB® expm function on an isomorphic matrix, and is therefore not accurate for larger matrices. The accuracy may be checked using the logm function which computes the inverse of the matrix exponential and should therefore yield a result of X to within rounding error.
