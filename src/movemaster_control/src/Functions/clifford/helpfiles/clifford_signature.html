Clifford Function Reference

clifford_signature

Initialises the Clifford Algebra.

Description

clifford_signature(p,q,r) initalises the signature of the Clifford Algebra being used. It must be called to initialise an algebra before any other multivector function can be called successfully. It accepts from zero to 3 parameters which must be non-negative integers. Any that are omitted default to zero. The three parameters represent the number of basis vectors in the algebra that square to +1, -1, and 0 respectively.

If called without parameters this function outputs a summary of the current alegbra if one has been initialised.

The function can also be called with one output parameter and zero input parameters. In this case it returns a 3-element row vector containing the signature of the current algebra (in the same order as the input parameters described above).

Examples

>> clifford_signature
Error using clifford_signature (line 72)
No Clifford algebra has been initialised.
 
>> clifford_signature(0,2)
>> clifford_signature
