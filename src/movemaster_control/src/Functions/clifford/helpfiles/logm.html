Clifford Function Reference

logm

Matrix logarithm
(Clifford overloading of standard MATLAB® function)

Syntax

X = logm(Y)

Description

logm(X) computes the matrix logarithm of a multivector matrix. The algorithm used is based on calling the MATLAB® logm function on an isomorphic matrix, and is therefore not accurate for larger matrices. It may also fail for some multivector matrices. The accuracy may be checked using the expm function.
