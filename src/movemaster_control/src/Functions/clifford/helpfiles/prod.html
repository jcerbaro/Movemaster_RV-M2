Clifford Function Reference

prod

Product of array elements
(Clifford overloading of standard MATLAB® function)

Syntax

Y = prod(X, dim)

Description

prod(X, dim) computes the product of the elements of the multivector array X along dimension dim.

The ordering within the product is important, because of the non-commutative clifford product: the product is computed with the lowest index on the left and the highest index on the right (that is, left to right for a row vector, or downwards for a column vector).

Examples

>> clifford_signature(0,2)
>> prod([e1, e2, e12])
 
ans = - 1.0000 e0
