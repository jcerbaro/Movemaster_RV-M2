Clifford Function Reference

log

Natural logarithm
(Clifford overloading of standard MATLAB® function)

Syntax

Y = log(X)

Description

log(X) computes the natural logarithm of the elements of the multivector array X.

Pending a better algorithm the current implementation uses the MATLAB® logm function on the isomorphic matrix of each element of X.
